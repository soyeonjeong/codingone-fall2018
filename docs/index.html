Coding I: Introduction to Web Development

Pratt Institute PMCG, Fall 2018

Instructor

So Yeon Jeong, sjeong18 [AT] gmail.com

Goal

The goal of this class is to teach basic web development programming skills so that you can have a web design and development portfolio piece to present to the world, as well as newfound CSS, HTML, JavaScript, JQuery skills to put on your resume. You should feel comfortable creating basic web projects after the class.

Assessment

  • Quizzes, 10%
  • Assignments, 25%
  • Participation, 25%
  • Final project, 40%

Final project: Website

  • Clear web design process
  • Multiple html pages
  • Including images and other multimedia
  • Full CSS style sheet
  • At least some minimal script (JavaScript/JQuery)
  • Presentation

Class structure

  1. Review of what we learned in the previous class (Occasional quiz), 30 min.
  2. Lecture, 1 hr.
  3. Break, 15 min.
  4. Pair programming on assignment based on what we learn in the lecture, 1 hr.

Syllabus

Week 1: The Fundamentals

Presentation

Topics

  • Course Overview
  • The Internet and World Wide Web
  • Introduction to Markup
  • Essential HTML for content
  • Essential HTML document structure

Assignments

  • Netscape 90s website recreation

Week 2: Markup, HTML

  • GitHub setup
  • Evaluating websites: content organization and usability
  • Web standards & accessability
  • Media content
  • How design drives action and organizes content

Week 3: HTML & CSS

  • HTML forms
  • CSS mechanics and basic selectors
  • CSS text properties

Assignments

  • Create HTML page with basic CSS base on image of web page

Week 4: Style (CSS)

  • Evaluating websites: style
  • CSS block modele
  • Colors and Images
  • CSS and lists
  • Multiple CSS stylesheets
  • The cascade and specificity of CSS

Week 5: Style CSS

  • Fonts, typography for web
  • Color design & theory: creating a color palette
  • CSS and Layouts
  • CSS Frameworks
  • Flexbox

Week 6: Style (CSS) and Images: Creating Web Graphics

  • CSS Loose Ends
  • Creating image types (PNG, JPEG, GIF, SVG), features, properties, and typical use
  • Scanning and optimizing web images
  • Icons
  • Tools for image creation and manipulation

Week 7: User Experience and Design

  • Workflow for designing a site
  • User Experience principles
  • Design principles
  • Wireframes
  • Information architecture
  • Mobile first, responsive design
  • Reading: Don't Make Me Think, Revisited: A Common Sense Approach to Web Usability by Steve Krug

Week 8: JavaScript: Introduction

  • JavaScript Events
  • DOM
  • jQuery
  • Reading: First 3 chapters of Learn jQuery: About jQuery, Using jQuery Core, and jQuery Events.

Week 9: Javascript: Integrating

  • Focusing on using and integrating JavaScript functionality
  • Slideshows, form validation, navigation, social media widgets

Week 10: Content Management Systems (CMS), Dynamic Languages

  • Intro to CMS
  • Open-source vs. private CMS
  • Other tools/languages for live website functionality (PHP, Ruby, SQL)

Week 11: Open class to work on projects & special topics (TBD)

Week 12: Final Project Presentations

  • Final project presentations!
